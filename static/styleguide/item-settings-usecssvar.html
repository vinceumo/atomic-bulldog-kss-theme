11.1 - Use CSS variable Flag

Source: settings/_atomic-bulldog-settings.scss, line 7

$use-css-var

By default Atomic Bulldog use CSS variables. You can compile the real values by changing this flag to false. It is not recommended to do so as it will break some theming options. Moreover postCSS can be used to compile CSS variables.

Can I Use css-variables? Data on support for the css-variables feature across the major browsers from caniuse.com.
